Duomi UI V1

多米通告 UI 高保真预览稿

这一版将多米通告定为“清爽可信的独立移动产品感”。 页面不再贴近微信工具页，而是更像一款成熟的垂直效率 App：有更完整的品牌层级、更顺滑的卡片分组、更强的首屏判断力，同时保留小程序场景所需的轻量效率。

视觉气质 可信、克制、现代、轻商务，适合合作撮合场景的专业感。
页面覆盖 共 18 个页面，按广场、发布方、达人、我的与治理四组统一出稿。
开发可用 配套 Token、组件规则、状态语义，可直接作为前端还原锚点。
判断更快 首页和详情页都把预算、平台、状态、CTA 拉到首屏，让报名决策更直接。
表单更轻 发布、报名、名片维护统一改成卡片分组和固定底部动作，不再是长表单堆叠。
治理不割裂 消息、举报、规则、反馈页继续保持同一产品语言，不像后台页面。
广告可兼容 后续广告仅以原生卡片位插入，不会破坏现有信息流骨架。
