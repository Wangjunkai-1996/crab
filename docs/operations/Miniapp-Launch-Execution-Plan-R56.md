## 4. 未来恢复 gate

以下动作继续保留为 future gate，不要求当前执行：

1. 在微信开发者工具导入真实 `AppID` 工程
2. 执行 `build-miniprogram-js.mjs`
3. 执行 `check-devtools-readiness.mjs`
4. 执行 `runtimeDebug.useCloud()`
5. 执行 `devSmoke.runFirstBatch()`
6. 回填 `miniprogram/evidence/r54/`

恢复时统一按 `MiniProgram-Real-Smoke-Runbook-R54.md` 执行，不另起新流程。
