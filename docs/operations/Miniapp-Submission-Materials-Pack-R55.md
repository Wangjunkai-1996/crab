## 3. 当前不能脑补的占位

以下内容必须由真实 owner 提供，当前不能自行代填：

1. 运营主体全称
2. 联系邮箱
3. 客服方式
4. 有管辖权的人民法院所在地
5. 发布 owner / 回滚 owner / 值守 owner 的具体人名与时间窗

## 4. R55 交付原则

1. 能直接复用现有文档的，先收口事实源，不重写第二份内容。
2. 没有真实值的字段只标 `待补`，不擅自填假信息。
3. 需要人工执行的动作统一后移，不让“没去点 DevTools”阻塞文档收口。
