## 3. 当前不能脑补的真实值

以下内容只能由真实 owner 提供，当前不得自行代填：

1. 运营主体全称
2. 联系邮箱
3. 客服方式
4. 有管辖权的人民法院所在地
5. 发布 owner / 回滚 owner / 值守 owner 的具体人名与时间窗

## 4. 交付原则

1. 复用既有事实源，不新增重复草案。
2. 没有真实值的字段继续标 `待补`。
3. 截图模板先固化规则，截图本身留到 future gate 恢复后再回填。
