# 多米通告 平台规则文案初稿 V1

## 说明

1. 本文档用于规则页、提审资料、审核后台规则和客服回复口径的统一。
2. 本规则与用户协议、隐私政策、审核与处罚 SOP 配套使用。

## 一、平台规则目标

本平台采用“低门槛发布、轻风控治理”的策略。为了在降低发布门槛的同时保障合作质量，平台将对通告发布、达人报名、资料展示、举报处理和违规处罚进行规则管理。

## 二、通告发布规则

### 2.1 基本要求

发布通告时，信息应真实、明确、可判断，至少应包含：

1. 合作平台
2. 合作领域
3. 城市或适用范围
4. 合作形式
5. 结算方式和预算区间或合作条件
6. 达人要求
7. 联系方式

### 2.2 推荐写法

推荐使用“做什么 + 找谁 + 在哪里 + 怎么合作”的表达方式，例如：

- 上海静安烘焙店招小红书探店达人，到店体验，可置换或小额付费
- 深圳美妆新品招寄拍达人，需提供图文或短视频案例

### 2.3 不予通过的常见情况

1. 只有一句“招达人合作”“加微信聊”
2. 标题、内容与实际合作无关
3. 缺少合作领域、结算方式、合作形式、城市等关键信息
4. 大量重复发布相似内容

## 三、达人报名规则

1. 达人资料应尽量真实、清晰、可判断。
2. 报名时应结合自身平台、领域、报价和案例情况如实提交信息。
3. 不得批量灌水报名、恶意占坑报名或使用虚假截图误导发布方。

## 四、平台禁止内容

以下内容或行为一经发现，平台有权驳回、下架、限制功能或封禁账号：

1. 虚假合作、诈骗合作、诱导收费
2. 押金、保证金、入群费、培训费等不合理收费
3. 冒充品牌、门店、机构、达人或第三方主体
4. 色情低俗、赌博诈骗、暴力恐怖、违法违规内容
5. 侵犯他人名誉、隐私、肖像、知识产权等合法权益
6. 垃圾广告、批量灌水、恶意导流
7. 恶意爬取、攻击、绕过平台审核和风控机制
8. 其他违反法律法规、微信小程序平台规则或公序良俗的内容

## 五、联系方式与隐私规则

1. 用户不得将通过平台获取的联系方式、社媒资料、案例截图等信息用于与合作无关的用途。
2. 未经对方明确同意，不得转卖、公开、传播其他用户资料。
3. 如用户利用平台资料进行骚扰、营销、诈骗或隐私泄露，平台有权直接限制或封禁账号。

## 六、举报与处罚规则

### 6.1 举报范围

用户可以针对以下对象发起举报：

1. 通告内容
2. 发布方资料
3. 达人资料
4. 骚扰、欺诈或恶意行为

### 6.2 处罚等级

平台将根据违规程度采取以下措施：

1. 警告
2. 驳回内容
3. 下架内容
4. 限制发布
5. 限制报名
6. 全量封禁

### 6.3 重点处罚情形

以下情形从严处理：

1. 已确认诈骗
2. 多次有效举报
3. 冒用主体
4. 恶意骚扰
5. 被处罚后继续规避规则

## 七、合作风险提示

1. 平台仅提供信息撮合服务，不提供合作履约担保。
2. 用户应在合作前自行核实对方身份、合作内容和履约能力。
3. 涉及线下见面、寄样、到店、报销、付款时，请用户自行注意安全并保留证据。

## 八、平台保留的管理权利

为维护平台秩序和用户体验，平台有权：

1. 审核、抽检、驳回、下架内容
2. 要求用户补充资料
3. 处理举报与申诉
4. 对违规账号采取限制措施
5. 根据法律法规或微信平台规则更新本规则

## 九、用户常见问答口径

### Q1：为什么我的通告没有通过？

可能原因包括：

1. 信息不完整
2. 内容表述过于模糊
3. 命中风险规则
4. 涉及平台禁止内容

### Q2：为什么我被限制发布或报名？

常见原因包括：

1. 多次发布低质或违规内容
2. 被多次有效举报
3. 资料存在明显伪造或冒用

### Q3：平台会保证合作一定成功吗？

不会。平台提供的是信息撮合与治理服务，不直接参与双方合作履行。

本规则建议同步到以下位置：

1. 小程序规则说明页
2. 发布页规则提示
3. 报名页风险提示
4. 举报页说明文案
5. 审核后台与客服回复口径
