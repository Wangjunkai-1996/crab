# 多米通告 隐私政策初稿 V1

## 重要提示

1. 本文档为产品内部起草版本，仅用于项目推进、页面文案和提审资料准备，不构成正式法律意见。
2. 上线前请结合实际功能、实际采集字段、第三方服务接入情况和法务意见进行复核。
3. 当前版本基于本项目已确认的 MVP 方案编写：低门槛发布、免费撮合、暂不接入会员付费和复杂广告系统。

## 一、我们是谁

`王俊凯`（以下简称“我们”）通过“多米通告”小程序向您提供达人合作信息发布、浏览、报名、审核、举报等服务。

我们非常重视您的个人信息和隐私保护，并将按照法律法规及本隐私政策处理您的个人信息。

## 二、本政策适用范围

本政策适用于您使用“多米通告”小程序及相关服务时，我们对您个人信息的收集、存储、使用、共享、转让、公开披露和保护。

本政策不适用于：

1. 其他用户基于站外合作自行处理您的信息
2. 第三方独立提供的服务或页面
3. 非由我们控制的第三方网站、产品或服务

## 三、我们可能收集的信息

### 3.1 您主动提交的信息

当您使用本平台时，您可能主动向我们提交以下信息：

1. 基础身份信息
   - 微信昵称、头像
   - 微信平台标识信息，如 OpenID、UnionID（如适用）
2. 发布方资料
   - 身份类型
   - 对外展示名称
   - 城市
   - 联系方式
   - 简介
3. 达人资料
   - 昵称
   - 城市
   - 擅长平台
   - 擅长领域
   - 粉丝量级
   - 社媒账号名称、账号 ID 或链接
   - 案例截图或作品截图
   - 联系方式
4. 通告内容
   - 通告标题
   - 平台、合作领域、城市、合作形式、结算方式、预算区间
   - 达人要求
   - 合作说明
   - 补充图片或截图
5. 报名信息
   - 自我介绍
   - 可提供的合作内容
   - 报价或合作条件
   - 补充案例材料
6. 举报与反馈信息
   - 举报原因
   - 补充说明
   - 上传凭证
   - 反馈内容

### 3.2 您使用服务时产生的信息

为保障服务正常运行、提升安全性和稳定性，我们可能收集以下日志和设备信息：

1. 设备信息
   - 设备型号
   - 操作系统版本
   - 小程序基础库版本
   - 网络状态
2. 日志信息
   - 访问时间
   - 页面访问记录
   - 操作日志
   - 异常崩溃日志
   - 审核、报名、举报等关键行为记录

### 3.3 我们不会在当前 V1 默认收集的内容

基于当前已确认产品范围，V1 默认不主动收集以下敏感能力对应信息，除非后续功能明确上线并另行告知您：

1. 支付信息
2. 精确位置信息
3. 通讯录
4. 相册或摄像头之外的额外系统权限
5. 广告个性化画像信息

若未来版本新增相关功能，我们会在功能上线前更新隐私政策并按要求获取相应授权。

## 四、我们如何使用您的信息

我们可能将您的信息用于以下目的：

1. 为您提供通告发布、展示、浏览、报名、审核、举报等核心服务
2. 帮助其他用户在合作场景下理解您的资料与需求
3. 进行内容审核、风险识别、违规治理、申诉处理
4. 保障服务安全、排查故障、分析运营情况
5. 处理用户投诉、反馈、举报和争议
6. 根据法律法规或监管要求履行法定义务

## 五、信息展示与共享规则

### 5.1 在平台内向其他用户展示的信息

为了完成信息撮合，部分信息会在平台内向其他用户展示：

1. 发布方公开展示信息
   - 对外展示名称
   - 身份类型
   - 城市
   - 通告内容
   - 资料完整度
2. 达人向发布方展示的信息
   - 昵称
   - 城市
   - 平台和领域信息
   - 粉丝量级
   - 案例截图
   - 报名介绍
   - 联系方式（在报名流程或合作判断需要的范围内）

### 5.2 我们可能共享信息的情形

在以下情况下，我们可能共享您的个人信息：

1. 在您明确同意或主动选择的范围内
2. 为实现本平台核心服务所必需，例如将报名资料展示给对应通告发布方
3. 与提供基础技术服务、云存储、信息安全服务的合作方在必要范围内共享
4. 根据法律法规、监管机关、司法机关要求进行提供

### 5.3 我们不会做的事情

除法律法规另有规定或您另行授权外，我们不会：

1. 向无关第三方出售您的个人信息
2. 将您的个人信息用于与本平台核心服务无关的用途
3. 在未充分告知的情况下引入与当前功能不匹配的大量第三方数据收集

## 六、我们如何存储和保护信息

1. 我们会采取合理必要的技术和管理措施保护您的信息安全，包括访问控制、权限管理、日志留痕、必要的数据隔离等。
2. 我们仅在实现本政策所述目的所必需的期限内保留您的个人信息，法律法规另有要求的除外。
3. 若发生个人信息安全事件，我们会按照法律法规要求及时采取补救措施，并视情况向您告知事件情况。

## 七、您如何管理自己的信息

您可以通过平台功能或联系我们的方式行使以下权利：

1. 查询和更正您的资料
2. 删除您主动发布的部分内容
3. 撤回部分授权
4. 注销账号或申请删除个人信息
5. 获取有关个人信息处理规则的说明

在以下情形下，您可以要求我们删除您的个人信息：

1. 我们违反法律法规处理您的个人信息
2. 我们超出约定目的收集或使用您的信息
3. 您不再使用本平台并申请注销
4. 法律法规规定的其他情形

## 八、关于未成年人

1. 若您为未成年人，请在监护人指导下阅读并使用本平台。
2. 若我们发现未成年人在未取得监护人同意的情况下提交了不适当信息，我们将尽快采取删除或其他合理处理措施。

## 九、第三方服务说明

### 9.1 微信平台能力

本平台运行在微信小程序环境中，部分基础能力由微信平台提供。您使用小程序时，还应遵守微信平台相关规则和协议。

### 9.2 云服务与基础技术服务

若本平台使用云开发、云存储、消息通知或其他基础技术服务，我们会在实现服务所必需的范围内处理相关数据。

### 9.3 未来新增广告或第三方服务

当前 V1 不以复杂广告系统为核心功能。若后续版本接入广告、统计或其他第三方服务，并涉及新增个人信息收集，我们会在上线前：

1. 更新本隐私政策
2. 完善小程序隐私保护指引
3. 依法取得必要授权

## 十、小程序隐私指引与授权说明

1. 我们会根据微信小程序平台规则，在小程序管理后台维护与实际功能相匹配的《用户隐私保护指引》。
2. 若未来版本调用微信平台隐私相关接口或组件，我们会按微信平台规则向您展示隐私协议或隐私授权提示，并在您同意后调用相应能力。

## 十一、本政策的更新

1. 我们可能根据法律法规、监管要求、产品功能变化或平台运营需要更新本隐私政策。
2. 若更新涉及重大变化，我们会通过页面公示、站内通知或其他合理方式提示您。
3. 更新后的政策一经公布生效；若您继续使用服务，视为您已阅读并同意更新后的政策。
